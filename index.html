Welcome to My Simple Webpage

This is a simple webpage designed with HTML and CSS.

Section 1: What is HyperText Markup Language (HTML)

HyperText Markup Language or HTML for short is a markup language which a document which could be displayed in any web browser.

Section 2: What can you do in HTML file?

In HTML file you can add unlimited line of codes to display everything on the webpage. You could also put CSS in the HTML as an option but recommended to make a css file along with the HTML file so you do not get confused what is HTML and CSS.

Section 3: What is Cascading Style Sheets (CSS)?

Cascading Style Sheets or CSS for short is a style sheet language used for describing the presentation of a document written in HTML.

Section 4: What can CSS do?

CSS can also create animation like sliding in from the left, right, down, up etc. can change the color of the text, the font size, the font style, the background color, the padding, the margin, the border, the border-radius, the box-shadow etc. There are a lot of animation that could make the webpage look interesting

Section 5: How is this webpage done?

This webpage is done by using HTML, and CSS. The HTML is used to create the structure of the webpage, and the CSS is used to style the webpage and animation as you could see that each of the section slides in from the left or the right except for the top which slides down and the footer which slides up.
